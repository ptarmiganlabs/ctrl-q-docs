# About Ctrl-Q

![Ctrl-Q logo](/logo/ctrl-q-transp.png){.about-logo}

Ctrl-Q makes life easier for Qlik Sense admins and developers.

It is a cross-platform command line tool for interacting with both client-managed Qlik Sense Enterprise on Windows and Qlik Sense Cloud.

It automates tedious, repetitive, and error-prone tasks related to Qlik Sense administration and development.

Ctrl-Q is open source and free to use and modify.

[Read the overview](/docs/overview/) to learn more.

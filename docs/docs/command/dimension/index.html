Delete master item dimensions in a Sense app

.\ctrl-q.exe master-item-dim-delete `
--host 192.168.100.109 `
--auth-user-dir LAB `
--auth-user-id goran `
--app-id a3e0f5d2-000a-464f-998d-33d333b175d7 `
--id-type name `
--master-item 'Country' 'Sales month'
